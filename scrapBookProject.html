How to Add a Memory

Follow these steps to add a new scrapbook memory to your collection.

  1. Click the "Add Memory" button to input your first memory
  2. Select a photo you want to upload
  3. Write a short description of the memory
  4. Add the details:
    • Input the exact date of the event
    • Tag the location if necessary
    • Click the "save" button
